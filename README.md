
  # Admin Terminal for NGO

 This is an human resource allocation app built to easily allocate volunteers in an NGO , monitor their works , efficiencey and assign them tasks in case of emergencies as well see their location and availability status and assign them tasks based on their skills and completion rate with a list of the total no. of tasks and priority list of tasks.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
